미니프로젝트
회원 로그인해서 회원 정보와 상품 정보를 관리

비로그인
1) 회원가입, 로그인, 상품목록  //done
2) 상품 상세 페이지에서 상품구입//done (상품을 구입하면 다음 페이지에서 상품 구입이 완료되었다고 표시하고 
					      재고수량을 1 감소시킴, 재고수량이 0이면 구매 불가)

로그인
1) 회원 정보 수정, 삭제  //done
2) 상품 입력 //done, 목록 //done, 수정//done 삭제(글쓴이만 수정,삭제 가능하게끔)//done


작업 순서
1) 환경 설정 (ojdbc6.jar 추가, context.xml 추가)
2) table 및 sequence 생성
3) 자바빈(DTO)
4) DAO
5) JSP


----필요한 체크----
1) 로그인 - 아이디 중복체크 // 재민
2) 상품 수정 - 작성자 비밀번호 확인(아마 DB수정 필요할듯) 
3) 상품 삭제 - 작성자 비밀번호 확인(아마 DB수정 필요할듯)
4) 재고수량이 0인지 체크 // 태형 -> done
5) 재고수량 - 구매수량이 0이상인지 체크  // 태형 -> done
6) 재고수량 - 구매수량 최대가 구매수량인지 체크 // 태형 -> done
7) 로그인 유무에 따라서 구매 가능하게 // 태형 -> done
