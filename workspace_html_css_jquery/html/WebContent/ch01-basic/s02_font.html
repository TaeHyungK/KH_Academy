눈길을 끄는 부분은 '붕괴3rd'가 기존의 흥행 공식을 따르지 않았다는 점이다. 일례로 이 게임에는 자동전투 기능이 없어 일일히 손으로 조작해야 한다. 상대해야 하는 적들의 난이도도 높아 한 판 한 판 신경써서 플레이해야 한다. 바쁜 직장인들을 겨냥해 자동전투 기능을 필수로 구현하는 국산 RPG들과는 전혀 다른 방향을 지향한 셈이다.
